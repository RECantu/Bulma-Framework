Intro to CSS

Author: Roman Cantu - 6-14-23

CSS, short for Cascading Style Sheets, is a programming language used for styling web pages. It describes how HTML elements should be displayed on a screen, paper, or other media. CSS allows you to control the appearance and layout of a website, including aspects such as colors, fonts, spacing, and positioning.

CSS3 is the latest version of CSS and introduced many new features and enhancements.Some notable features include rounded corners, gradients, shadows, transformations, transitions, animations, and flexible box layout (Flexbox) and grid layout (CSS Grid) for advanced page layout.

CSS is continuously evolving, with new features and specifications being added over time. It plays a crucial role in modern web development, allowing developers to create visually engaging and user-friendly websites.
